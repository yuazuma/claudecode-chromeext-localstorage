# 要件定義書: テキストエリアバックアップ拡張機能

## 概要

Chrome / Edge 対応のブラウザ拡張機能。ページ内のテキスト入力要素への入力内容を自動・手動でバックアップし、後から復元できる。

---

## 対象要素

| 要素                       | 備考                             |
| -------------------------- | -------------------------------- |
| `<textarea>`               | 標準テキストエリア               |
| `<input type="text">`      | 1行テキスト入力                  |
| `contenteditable` 属性要素 | Gmail・Notion 等のリッチエディタ |

---

## バックアップ動作

### 自動バックアップ

- **トリガー**: 対象要素からフォーカスが外れたとき（`blur` イベント）
- 内容が空の場合は保存しない
- 前回保存内容と同一の場合はスキップ

### 手動バックアップ

- ポップアップ上の「今すぐ保存」ボタンから実行
- 現在フォーカスされている要素、または全要素をまとめて保存

---

## データ識別キー

各バックアップは以下の組み合わせで識別する:

```
{ページURL（クエリなし）} + {CSSセレクター or 要素ID}
```

---

## 保持ポリシー（設定画面から変更可能）

| 設定項目               | デフォルト値 |
| ---------------------- | ------------ |
| 要素ごとの最大保持件数 | 50 件        |
| 最大保持期間           | 30 日        |

いずれかを超えた場合、古いものから自動削除。

---

## サイト有効/無効

- **管理単位**: オリジン（例: `https://example.com`）
- デフォルトはすべてのサイトで**有効**
- ポップアップのトグルで即時切り替え
- 設定画面でサイト一覧を管理

---

## ストレージ

- `chrome.storage.local` を使用
- 推定容量: 1 バックアップあたり最大 ~10KB、上限 10MB を目安に警告

---

## ポップアップ UI

| 要素                  | 説明                                                         |
| --------------------- | ------------------------------------------------------------ |
| サイト有効/無効トグル | 現在のオリジンを即時切り替え                                 |
| 「今すぐ保存」ボタン  | フォーカス中の要素を手動保存                                 |
| バックアップ一覧      | 現在のページに紐づく候補を表示。クリックで対象要素にリストア |
| 設定ページリンク      | オプションページを開く                                       |

---

## 設定画面（オプションページ）

| 機能                   | 詳細                                            |
| ---------------------- | ----------------------------------------------- |
| バックアップ一覧・閲覧 | サイト/要素ごとにフィルタリング、内容プレビュー |
| 手動削除               | 個別削除・全削除                                |
| サイト管理             | 有効/無効の一覧、サイト単位で削除               |
| エクスポート           | JSON 形式でダウンロード                         |
| インポート             | JSON ファイルから復元（マージまたは上書き）     |
| 保持ポリシー設定       | 最大件数・保持期間を変更                        |

---

## 必要な権限

```json
"permissions": ["storage", "activeTab", "scripting"],
"host_permissions": ["<all_urls>"]
```

---

## 非機能要件

- キーボードショートカット: なし
- クラウド同期（chrome.storage.sync）: なし（ローカルのみ）
- Manifest V3（MV3）準拠
